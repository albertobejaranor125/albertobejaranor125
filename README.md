## About Me

I am a Computer Engineer holding a Master’s Degree in Video Game Development. My profile bridges solid software engineering fundamentals with interactive system development, working heavily with **C# (.NET), C++, and Python**. 

With hands-on experience in corporate **DevSecOps (Accenture)** and generalist software development, I focus on clean code, system modularity, automation pipelines, and high-performance gameplay systems. I love taking prototypes from concept to production.

* 🎮 **Playable builds available on:** [My itch.io Portfolio](https://albertobejaranor.itch.io/)

## Featured Projects

* 🤖 **The Revenge of Pepai:** An arcade action-platformer built in Unity/C#. Features an agile movement system (dashing, combo chaining) and an event-driven scoring system, demonstrating optimized gameplay loops and responsive character controllers.
* 🧠 **MindFeedNees:** A serious game built in Unity/C# aimed at supporting depression diagnosis in people with morbid obesity. Developed with a focus on stable UI systems and data tracking.
* ⏳ **Unity20thAnniversaryJam:** A time-travel themed prototype developed under tight deadlines for the official Unity 20th Anniversary Jam, showcasing rapid mechanic implementation and modular architecture.
